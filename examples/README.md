# examples

本目录存放示例输入、参数文件和本地测试素材。

## 当前主要内容

| 文件/目录 | 作用 |
| --- | --- |
| `plate_params.json` | SolidWorks 通用带孔矩形板示例参数。 |
| `wing_params.json` | 通用低速无人机/教学用途梯形机翼示例参数。 |
| `vision_test/` | 视觉模型测试图片目录。 |

## 维护规则

- 示例文件用于演示和测试，不应包含真实 API Key、账号、密码或隐私信息。
- 建模输出不要放在这里，应统一进入 `generated_models/`。
- 图片测试素材不得包含敏感武器结构或隐私内容。
- 修改示例参数后，应同步检查对应脚本是否还能读取。
