# domain_docs

本目录存放人工整理的 Markdown 知识资料。这里的内容会进入知识库索引，并通常被标记为 `curated_domain_note`，适合用来沉淀稳定术语、流程、模板和高层知识。

## 子目录

| 目录 | 作用 |
| --- | --- |
| `00_core_subjects/` | 核心领域知识文件。 |
| `01_literature_notes/` | 文献摘要卡片。 |
| `02_textbook_notes/` | 教材笔记。 |
| `03_writing_templates/` | 写作模板。 |
| `04_engineering_workflows/` | 工程流程说明、知识库使用策略和检查清单。 |
| `05_glossary/` | 术语表、缩略语和检索词说明。 |

## 适合放入的内容

- 人工总结后的知识卡片。
- 公开资料的非敏感摘要。
- 写作模板、综述提纲、报告结构。
- 检索词、术语表和中英文对应关系。
- 知识库使用策略、安全边界和证据追溯规则。

## 不建议放入的内容

- 原始 PDF、Word、压缩包，应该放入 `raw_docs/`。
- 自动提取文本，应该放入 `processed_docs/`。
- 长期记忆或用户偏好，应该放入 `memory/`。
- API Key、账号、密码、token。
