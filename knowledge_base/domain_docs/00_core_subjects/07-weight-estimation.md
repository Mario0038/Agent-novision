# 重量估算

## 1. 起飞重量方程

飞行器起飞重量由以下部分组成：
```
W_TO = W_empty + W_payload + W_fuel
     = W_empty + W_payload + W_fuel_mission + W_fuel_reserve
```

### 1.1 基本迭代公式

空机重量通常用起飞重量的比例表示：
```
W_empty / W_TO = a × W_TO^b
```
其中 a, b 为统计常数（随飞机类型变化）。

### 1.2 迭代求解

```
W_TO = W_payload / (1 - W_fuel/W_TO - W_empty/W_TO)
```

由于 W_empty/W_TO 是 W_TO 的函数，需迭代求解，通常 3~5 次收敛。

## 2. 空机重量估算

### 2.1 Raymer 统计公式

| 飞机类型 | a | b |
|----------|---|---|
| 通用飞机（单发） | 1.20 | -0.10 |
| 双发涡桨 | 0.96 | -0.05 |
| 喷气运输机 | 0.92 | -0.07 |
| 军用教练机 | 1.59 | -0.10 |
| 战斗机 | 2.34 | -0.13 |
| 无人机（复合材） | 0.85 | -0.05 |

公式形式：
```
W_empty/W_TO = a × W_TO^b
```

### 2.2 部件重量分解

| 部件 | 占比（典型值） | 估算方法 |
|------|---------------|----------|
| 机翼 | 8%~12% | 基于翼面积、展弦比、载荷因子 |
| 机身 | 8%~12% | 基于浸润面积、长度 |
| 尾翼 | 2%~3% | 基于尾翼面积 |
| 起落架 | 3%~5% | 基于起飞重量 |
| 动力系统 | 8%~15% | 基于发动机干重 |
| 飞控/航电 | 2%~5% | 基于功能清单 |
| 固定设备 | 5%~8% | 统计比例 |

### 2.3 机翼重量估算

```
W_wing = 0.0051 × (W_TO × n_max)^0.557 × S_w^0.649 × AR^0.5
         × (t/c)_root^(-0.4) × (1+λ)^0.1 × cos(Λ)^(-1)
```

（Raymer 通用航空公式，单位为磅和英尺）

## 3. 燃油重量估算

### 3.1 任务燃油迭代

```
W_fuel = W_TO × (1 - Π M_ff_i)
```

其中 M_ff_i 是各任务段的终点重量/起点重量比。

### 3.2 典型燃料比

各阶段的燃料消耗比例：
```
发动机启动&暖机：M_ff = 0.990~0.995
滑行：M_ff = 0.990~0.995
起飞：M_ff = 0.970~0.985
爬升：M_ff = 0.980~0.990
巡航：M_ff = exp(-R × TSFC / (V × L/D))
待机：M_ff = exp(-E × TSFC / (L/D))
下降：M_ff = 0.990~0.995
着陆：M_ff = 0.992~0.997
```

### 3.3 余油要求
- VFR 日间：30 分钟巡航余油
- IFR：45 分钟巡航余油 + 备降机场
- 军用：5% 总燃油 + 作战消耗

## 4. 重心估算

### 4.1 重心位置
```
x_cg = Σ (W_i × x_i) / Σ W_i
```

### 4.2 重心包线
重心随燃油消耗、载荷变化而移动，需保证在所有装载状态下：
```
x_cg_forward ≤ x_cg ≤ x_cg_aft
```

### 4.3 重心管理
- 燃油顺序消耗策略
- 前后货舱分配
- 配重（最后手段）

## 5. 惯量估算

### 5.1 简化估算
```
I_xx ≈ W/g × (b/2)² × k_x²  (k_x ≈ 0.2~0.25)
I_yy ≈ W/g × (L/2)² × k_y²  (k_y ≈ 0.3~0.4)
I_zz ≈ W/g × (L/2)² × k_z²  (k_z ≈ 0.3~0.4)
```

## 6. 重量增长控制

设计过程中重量倾向于增长（"重量蠕变"），控制策略：
1. 所有部件设定重量目标并定期称重
2. 每阶段保留 5% 设计余量
3. 超出目标的部件强制减重
4. 复合材料替代金属是最直接手段
