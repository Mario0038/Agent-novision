# 飞行器总体设计概论

## 1. 总体设计的定义与地位

飞行器总体设计（Aircraft Conceptual/Preliminary Design）是飞行器研制过程中
最顶层的设计活动，负责将用户需求转化为飞行器的总体技术方案。总体设计决定了
飞行器 80% 以上的全寿命周期成本和技术性能。

### 设计阶段划分

| 阶段 | 名称 | 主要工作 | 输出 |
|------|------|----------|------|
| Phase I | 概念设计 | 方案探索、参数估算 | 概念方案 |
| Phase II | 初步设计 | 方案细化、风洞试验 | 初步方案 |
| Phase III | 详细设计 | 详细图纸、强度校核 | 生产图纸 |

## 2. 总体设计的核心矛盾

飞行器设计是一个多学科耦合的复杂系统工程，核心矛盾包括：

- **重量 vs 强度**：结构越轻越易超载损坏
- **升力 vs 阻力**：增升往往伴随增阻
- **性能 vs 成本**：高性能 = 高成本
- **静稳定性 vs 机动性**：越稳定越不灵活
- **航程 vs 载荷**：燃油越多航程越长但载荷越小

## 3. 设计哲学

### 3.1 迭代设计思想
总体设计不是一次性完成的，而是通过"估算—分析—修正—再估算"的螺旋上升过程。

### 3.2 权衡研究（Trade Study）
对于存在矛盾的设计参数，需要通过权衡研究找到最优平衡点。
常用方法：
-  carpet plot（毯式曲线）
-  多目标优化（MDO）
-  灵敏度分析

### 3.3 设计余量原则
所有关键参数均应预留设计余量（通常 5%~15%），以应对后续设计深化
带来的重量增长和性能衰减。

## 4. 现代设计工具与方法

- **CAD/CAE 集成**：CATIA、NX 等三维设计
- **CFD 分析**：Fluent、CFX 等气动计算
- **FEM 分析**：Nastran、ANSYS 等结构分析
- **MDO 框架**：ModelCenter、Isight 等多学科优化
- **MBSE**：基于模型的系统工程（SysML/UML）

## 5. 适航与标准

飞行器设计必须满足适航要求：
- 民用：CCAR-23（正常类）、CCAR-25（运输类）
- 军用：GJB 系列军用标准
- 无人机：CCAR-92 无人机适航管理规定
