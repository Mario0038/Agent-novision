# 多学科设计与优化（MDO）

## 1. MDO 基本概念

多学科设计优化（Multidisciplinary Design Optimization）是处理飞行器设计
中各学科间耦合关系的系统方法。

### 1.1 飞行器设计中的耦合关系

```
气动 ←→ 结构 ←→ 动力
  ↕      ↕      ↕
 重量 ←→ 控制 ←→ 隐身
```

### 1.2 耦合类型
- **强耦合**：气动→结构（气动载荷决定结构应力）
- **中度耦合**：动力→重量（发动机选型决定重量）
- **弱耦合**：隐身→控制（隐身外形间接影响操控面效率）

## 2. MDO 优化方法

### 2.1 单级优化（AAO — All-At-Once）
所有学科变量同时优化
- 优点：理论上可找到全局最优
- 缺点：问题规模大，计算量爆炸

### 2.2 多级优化（CO — Collaborative Optimization）
```
系统级优化器
  ├── 气动学科优化器
  ├── 结构学科优化器
  ├── 动力学科优化器
  └── 控制学科优化器
```
- 优点：学科解耦，并行计算
- 缺点：收敛性问题（CO特有）

### 2.3 BLISS 方法
```
系统级：优化少量全局变量（翼面积、展弦比等）
学科级：优化局部变量（蒙皮厚度、翼型等）
迭代：系统→学科→系统→...
```
- 优点：收敛性好，效率高

### 2.4 代理模型方法
用响应面/Kriging/神经网络替代高精度分析，加速MDO

## 3. 常见优化问题类型

### 3.1 MTOW 最小化
```
目标：min W_TO
约束：所有性能满足TLARs
设计变量：AR, S, T/W, t/c, ...
```

### 3.2 航程最大化
```
目标：max R
约束：W_TO ≤ W_TO_max
设计变量：AR, S, W_fuel, ...
```

### 3.3 成本最小化
```
目标：min LCC（全寿命周期成本）
约束：性能满足底线要求
```

### 3.4 多目标优化
Pareto 前沿方法：在互相矛盾的目标间寻找最优折衷面
- 航程 vs 重量
- 性能 vs 成本
- 隐身 vs 气动效率

## 4. 灵敏度分析

### 4.1 设计参数对性能的敏感度
```
∂R / ∂AR → 展弦比对航程的影响
∂W_TO / ∂S → 翼面积对起飞重量的影响
```

### 4.2 灵敏度矩阵
```
S_ij = ∂f_i / ∂x_j
```
其中 f_i 是第i个响应量，x_j 是第j个设计变量。

## 5. 经典矛盾与权衡

### 5.1 翼载荷 W/S 的权衡
- 低翼载荷：起飞着陆性能好，机动性好 → 翼面积大，重量大，阻力大
- 高翼载荷：巡航效率高，结构轻 → 起飞着陆距离长，机动性差

### 5.2 展弦比 AR 的权衡
- 高展弦比：诱导阻力小，升阻比高 → 结构重，滚转惯性大
- 低展弦比：结构轻，机动性好 → 诱导阻力大

### 5.3 推重比 T/W 的权衡
- 高推重比：起飞/爬升/机动性能好 → 发动机重，耗油多
- 低推重比：巡航效率高，结构轻 → 起飞性能可能不达标

## 7. MDO 实施建议

1. **从简单开始**：先用参数化模型 + 经验公式，再逐步替代为高精度分析
2. **分阶段优化**：概念设计用粗粒度MDO，详细设计用细粒度MDO
3. **人工干预**：MDO不替代设计师判断，而是提供定量依据
4. **不确定性量化**：不只看最优解，还要分析最优解的鲁棒性
5. **可视化展示**：Pareto前沿、灵敏度饼图、耦合强度矩阵
