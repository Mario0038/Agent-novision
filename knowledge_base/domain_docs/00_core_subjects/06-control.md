# 飞控系统

## 1. 操控面布局

### 1.1 主要操控面
| 操控面 | 功能 | 位置 |
|--------|------|------|
| 副翼（Aileron） | 滚转控制 | 机翼外侧后缘 |
| 升降舵（Elevator） | 俯仰控制 | 水平尾翼后缘 |
| 方向舵（Rudder） | 偏航控制 | 垂直尾翼后缘 |
| 全动平尾（Stabilator） | 俯仰控制 | 水平尾翼整体偏转 |
| 扰流板（Spoiler） | 滚转+减速 | 机翼上面 |

### 1.2 操控面面积配置
- 副翼面积：机翼面积的 5%~8%
- 副翼展长：半翼展的 30%~50%
- 升降舵面积：平尾面积的 25%~40%
- 方向舵面积：垂尾面积的 30%~45%

### 1.3 特殊操控面
- **升降副翼（Elevon）**：飞翼布局，同时进行俯仰和滚转控制
- **方向升降舵（Ruddervator）**：V型尾翼，同时进行俯仰和偏航控制
- **前翼（Canard）**：鸭式布局俯仰控制
- **推力矢量（Thrust Vectoring）**：发动机喷口偏转辅助控制

## 2. 静稳定性

### 2.1 纵向静稳定性
```
C_mα < 0 （负梯度 = 稳定）
```
- 静稳定裕度 SM = 5%~15% MAC（常规）
- 放宽静稳定（RSS）：SM ≤ 0，需增稳系统

### 2.2 横向静稳定性
```
C_lβ < 0 （负梯度 = 稳定）
```
- 上反角为正贡献
- 后掠角为正贡献
- 高置翼为正贡献

### 2.3 航向静稳定性
```
C_nβ > 0 （正梯度 = 稳定）
```
- 垂尾面积是主要贡献
- 机身后体是负贡献（破坏性）

## 3. 飞控系统架构

### 3.1 机械操纵系统
- 拉杆/钢索 + 连杆 + 摇臂
- 简单可靠，有人机的基本配置
- 存在摩擦、间隙、弹性变形

### 3.2 电传飞控（Fly-by-Wire, FBW）
- 传感器 → 飞控计算机 → 作动器
- 可实现增稳、包线保护、自动配平
- 军民用先进飞机标配
- 余度要求：至少三余度（军机）/ 双余度（民机）

### 3.3 光传飞控（Fly-by-Light）
- 光纤替代电缆
- 抗电磁干扰能力强

### 3.4 自主控制（无人机）
层级架构：
```
任务规划层（路径、策略）
    ↓
导航层（位置、姿态、速度估计）
    ↓
控制分配层（舵面、油门分配）
    ↓
执行层（作动器、发动机）
```

## 4. PID控制（基础）

```
δ = K_p × e + K_i × ∫e dt + K_d × de/dt
```

无人机常用控制回路：
- **姿态回路**（内环）：角速率控制，频率最高（50~200 Hz）
- **姿态角回路**（中环）：姿态角控制
- **导航回路**（外环）：位置/航迹控制，频率最低（1~10 Hz）

## 5. 传感器

| 传感器 | 测量量 | 精度要求 |
|--------|--------|----------|
| IMU | 角速率、加速度 | 0.1°/h（战术级） |
| GPS/GNSS | 位置、速度 | 1~5 m |
| 空速管 | 空速、气压高度 | ±1 m/s |
| 磁力计 | 航向 | ±1° |
| 激光雷达 | 避障 | ±5 cm |

## 6. 冗余设计

- 三余度飞控计算机 + 多数表决
- 双余度电源
- 作动器故障安全模式（断电→回中或阻尼）
- 通信链路中断的自主返航能力
