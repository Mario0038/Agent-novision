# 需求分析与指标分解

## 1. 需求来源

飞行器设计需求通常来自：
- **作战需求**（军用）：打击、侦察、运输、制空等
- **运营需求**（民用）：客运、货运、通航、公务等
- **特殊需求**：消防、农业、测绘、救援等

## 2. 任务剖面（Mission Profile）

任务剖面是飞行器在一次典型任务中各阶段的时间-状态关系图。

### 典型任务剖面分段

```
起飞 → 爬升 → 巡航 → 待机/任务 → 下降 → 进场着陆
```

对于军用飞机，还需考虑：
- 作战半径 = 出航 + 任务 + 返航 + 余油
- 任务段通常包含：突防、目标搜索、武器投放、退出

### 任务剖面绘制要求

必须明确各段的：
1. 高度
2. 速度（马赫数或当量空速）
3. 时间或航程
4. 重量变化（燃油消耗、武器投放）
5. 特殊要求（隐身、低空突防等）

## 3. 顶层设计要求（TLARs）

TLARs（Top-Level Aircraft Requirements）是设计的直接输入：

### 3.1 性能指标
| 指标 | 符号 | 单位 |
|------|------|------|
| 最大速度 | V_max | km/h 或 Ma |
| 巡航速度 | V_cruise | km/h |
| 失速速度 | V_stall | km/h |
| 最大航程 | R_max | km |
| 最大航时 | T_max | h |
| 实用升限 | H_ceiling | m |
| 起飞距离 | S_TO | m |
| 着陆距离 | S_Land | m |
| 最大爬升率 | ROC_max | m/s |
| 最大过载 | n_max | g |

### 3.2 重量指标
- 有效载荷（Payload）
- 空机重量约束
- 最大起飞重量限制

### 3.3 几何约束
- 翼展限制（如机库门宽度）
- 机长限制
- 折叠要求

## 4. 需求分解方法

### 4.1 QFD（质量功能展开）
将用户需求逐层分解为技术指标，建立需求-指标关系矩阵。

### 4.2 指标权衡
- 航程 vs 载荷：Breguet 方程表明航程与 ln(W_initial/W_final) 成正比
- 速度 vs 效率：速度越高，阻力越大，升阻比越低
- 隐身 vs 气动：隐身外形往往与气动效率矛盾

## 5. 约束分析

### 5.1 机场适应性
- 跑道长度限制 → 起飞着陆距离约束
- 道面承载力 → 轮胎压强限制

### 5.2 使用环境
- 高寒/高温/高原适应性
- 腐蚀环境（海上）
- 沙尘环境（沙漠）

### 5.3 保障条件
- 可用的维护设备
- 备件供应链
- 人员技术水平

## 6. 需求验证

需求确认后需进行验证：
- 是否可度量？
- 是否可实现？
- 是否存在矛盾？
- 是否有遗漏？
