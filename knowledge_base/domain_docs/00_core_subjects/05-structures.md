# 结构设计

## 1. 结构设计原则

### 1.1 基本要求
- **强度**：承受极限载荷而不破坏（安全系数 1.5）
- **刚度**：变形控制在气动弹性允许范围内
- **疲劳**：满足设计使用寿命（起落次数/飞行小时）
- **损伤容限**：裂纹在检测前不扩展到临界尺寸
- **轻重量**：结构重量占比最小化

### 1.2 结构重量占比统计
| 飞机类型 | 结构重量/起飞重量 |
|----------|-------------------|
| 轻型通用飞机 | 35% ~ 45% |
| 支线客机 | 30% ~ 40% |
| 大型运输机 | 25% ~ 35% |
| 战斗机 | 25% ~ 35% |
| 长航时无人机 | 20% ~ 30% |
| 小型无人机 | 15% ~ 25% |

## 2. 材料选择

### 2.1 铝合金
- 2024-T3：高强度，疲劳性能好（机身蒙皮）
- 7075-T6：超高强度（翼梁、隔框）
- 6061-T6：焊接性好，耐腐蚀（次要结构）

### 2.2 钛合金
- Ti-6Al-4V：高温区结构，发动机周边
- 密度为钢的60%，强度相当

### 2.3 复合材料
- **碳纤维/环氧**（CFRP）：
  - E = 70~180 GPa（模量可变）
  - σ = 600~2500 MPa
  - 密度 ρ ≈ 1600 kg/m³
  - 可减重 15%~25%
- **玻璃纤维/环氧**（GFRP）：次承力结构、整流罩
- **芳纶纤维/环氧**（KFRP）：抗冲击、雷达罩

### 2.4 增材制造
- 铝合金（AlSi10Mg）SLM 成形
- 钛合金（Ti-6Al-4V）EBM/SLM 成形
- 拓扑优化 + 增材制造 = 可减重 30%~50%

## 3. 机翼结构

### 3.1 典型结构形式
```
蒙皮 — 长桁 — 翼肋 — 翼梁 — 翼身接头
```

### 3.2 受力分配
- 翼梁承受弯矩（翼根弯矩最大）
- 蒙皮承受扭矩（闭合剖面扭转刚度）
- 翼肋维持翼型形状、传递分布载荷

### 3.3 翼梁设计
```
M_root ≈ (n_max × W_TO × g × b) / (8 × safety_factor)
```
翼梁高度由翼型厚度决定：h_spar ≈ 0.6~0.8 × t_airfoil × c

## 4. 机身结构

### 4.1 结构形式
- **桁架式**：钢管焊接骨架 + 蒙皮（轻型飞机）
- **半硬壳式**：隔框 + 长桁 + 蒙皮（主流）
- **硬壳式**：纯蒙皮承力（极少使用）
- **夹层结构**：复合材料蜂窝夹层

### 4.2 机身载荷
- 气动载荷（分布载荷）
- 惯性载荷（集中质量：发动机、载荷、燃油）
- 增压载荷（客机）
- 着陆冲击载荷

## 5. 起落架设计

### 5.1 布局形式
- **后三点式**：主轮在前，尾轮在后（草地跑道友好）
- **前三点式**：前轮 + 双主轮（主流，方向稳定性好）
- **自行车式**：前后单轮 + 翼尖撑杆（高速飞机）

### 5.2 起落架位置
- 前三点：主轮在重心后 10%~15% MAC
- 前轮载荷：8%~15% 起飞重量
- 需满足防倒立角、防侧翻角要求

### 5.3 缓冲器
- 油气式缓冲器：效率 80%~90%
- 弹簧式：简单可靠
- 着陆下沉速度（设计值）：3 m/s（陆基）、4~5 m/s（舰载）

## 6. 颤振与气动弹性

### 6.1 颤振
机翼弯扭耦合自激振动，需保证颤振速度 V_F > 1.2 × V_D（设计俯冲速度）

### 6.2 发散
前掠翼发散问题更严重，需增加扭转刚度

### 6.3 操纵反效
副翼偏转导致机翼扭转过大而反向滚转
