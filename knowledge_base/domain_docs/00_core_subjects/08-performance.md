# 性能分析

## 1. 起飞性能

### 1.1 起飞距离
```
S_TO = S_GR (地面滑跑) + S_TR (过渡) + S_CL (越障)
```

地面滑跑距离简化式：
```
S_GR = (1.21 × W/S) / (g × ρ × C_L,max × T/W)
```

### 1.2 关键参数
- V_stall（失速速度）：
```
V_stall = sqrt(2 × W/S / (ρ × C_L,max))
```
- V_R（抬轮速度）：≥ 1.05 × V_stall
- V_LOF（离地速度）：≥ 1.1 × V_stall
- V_2（起飞安全速度）：≥ 1.2 × V_stall

### 1.3 起飞性能影响因素
- 场高（密度高度）：高温、高原 → 性能衰减
- 跑道坡度：上坡增加滑跑距离
- 跑道表面：草地/湿跑道增加摩擦
- 风速：逆风有利，顺风不利

## 2. 爬升性能

### 2.1 爬升率
```
ROC = (T - D) × V / W = V × (T/W - 1/(L/D))
```

### 2.2 最佳爬升速度
- 最快爬升：接近 V_Y（最佳爬升速度）
- 最陡爬升：接近 V_X（最佳爬升角速度）
- 喷气机 V_Y ≈ V for max (T-D)/W
- 螺旋桨 V_Y ≈ V for max excess power

### 2.3 升限
- 实用升限：ROC = 0.5 m/s
- 绝对升限：ROC = 0 m/s

## 3. 巡航性能

### 3.1 Breguet 航程方程（螺旋桨）
```
R = (η_prop / c_P) × (L/D) × ln(W_i / W_f)
```

### 3.2 Breguet 航程方程（喷气）
```
R = (V / c_T) × (L/D) × ln(W_i / W_f)
```

### 3.3 最大航程条件
- 螺旋桨：飞在最大 L/D 状态
- 喷气：飞在最大 V×L/D 状态

### 3.4 巡航高度优化
- 随燃油消耗、重量减轻，逐步爬升（阶梯巡航）
- 最佳巡航高度接近最佳升阻比对应的升力系数

## 4. 航时（留空时间）

### 4.1 Breguet 航时方程（螺旋桨）
```
E = (η_prop / c_P) × (L/D) × (C_L^1.5 / C_D) × sqrt(2ρS)
     × (W_f^{-0.5} - W_i^{-0.5})
```

### 4.2 Breguet 航时方程（喷气）
```
E = (1 / c_T) × (L/D) × ln(W_i / W_f)
```

### 4.3 最大航时条件
- 螺旋桨：飞在最小功率（C_L^1.5/C_D 最大）
- 喷气：飞在最大 L/D

## 5. 着陆性能

### 5.1 着陆距离
```
S_Land = S_A (进近) + S_FL (拉平) + S_FR (自由滑跑) + S_BR (刹车)
```

### 5.2 着陆速度
- V_REF（进场速度）：≥ 1.3 × V_stall_landing
- V_TD（接地速度）：≥ 1.15 × V_stall_landing

### 5.3 着陆性能影响因素
- 着陆重量（越轻越好，但需满足复飞推力要求）
- 襟翼设定（全襟翼降低接地速度，但增加阻力）
- 反推/刹车效率
- 跑道状态

## 6. 机动性能

### 6.1 持续转弯性能
```
n = 1 / cos(φ) = L / W
ω = g × sqrt(n² - 1) / V
R = V² / (g × sqrt(n² - 1))
```

### 6.2 能量机动性（战斗机）
```
P_s = (T - D) × V / W  （单位剩余功率）
```

### 6.3 V-n 图（飞行包线）
- 正过载限制（通常 +2.5 ~ +9g）
- 负过载限制（通常 -1.0 ~ -3g）
- 机动速度 V_A（全偏操纵面不失速的最大速度）
- 设计俯冲速度 V_D

## 7. 飞行包线

完整的飞行包线包含：
1. **高度-速度包线**：最大/最小速度 vs 高度
2. **V-n 包线**：过载 vs 速度（含突风载荷线）
3. **重量-重心包线**：重量 vs 重心位置
