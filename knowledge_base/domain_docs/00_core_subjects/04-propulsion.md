# 动力系统选型

## 1. 动力装置类型

### 1.1 活塞发动机
- 功率范围：10~500 HP
- 适用速度：< 300 km/h
- 适用高度：< 6000 m
- 耗油率 SFC：0.25~0.35 kg/(kW·h)
- 优点：成本低、维护简单
- 缺点：功率重量比低、高空性能差
- 典型应用：轻型飞机、小型无人机

### 1.2 涡桨发动机
- 功率范围：500~10000 SHP
- 适用速度：300~700 km/h
- 适用高度：< 10000 m
- 耗油率 SFC：0.28~0.35 kg/(kW·h)
- 优点：燃油效率高、中低速推力好
- 缺点：速度上限低
- 典型应用：支线客机、中空长航时无人机（MQ-9）

### 1.3 涡扇发动机
- 推力范围：10~500 kN
- 适用速度：M 0.3~0.9（高涵道比）/ M 0~2.5（低涵道比）
- 适用高度：< 15000 m
- 耗油率 TSFC：0.35~0.6 kg/(daN·h)（高涵道比）
- 优点：效率高、推力大
- 缺点：成本高、响应慢
- 典型应用：客机、运输机、高空长航时无人机（Global Hawk）

### 1.4 涡喷发动机
- 推力范围：1~200 kN
- 适用速度：M 0.5~3+
- 耗油率 TSFC：0.8~1.2 kg/(daN·h)
- 优点：高速性能好
- 缺点：效率低
- 典型应用：战斗机、靶机、高速无人机

### 1.5 电动机
- 功率范围：< 1 ~ 500 kW（航空级）
- 优点：零排放、低噪音、响应快
- 缺点：能量密度低（电池 ~250 Wh/kg vs 航空煤油 ~12000 Wh/kg）
- 典型应用：小型多旋翼、eVTOL、城市空中交通

### 1.6 混合动力
- 串联：发动机→发电机→电动机→螺旋桨
- 并联：发动机+电动机→变速箱→螺旋桨
- 优点：兼顾航程与排放
- 缺点：系统复杂度高

## 2. 动力选型决策流程

```
任务剖面分析 → 功率/推力需求 → 工作高度与速度范围
                                    ↓
           燃油/能量类型选择 → 动力类型匹配
                                    ↓
                  发动机型号初选 → 性能校核
```

## 3. 推重比/功重比估算

### 3.1 统计公式（初步估算）

**起飞推重比**：
```
T/W = a × (W_TO/S)^c × (V_max)^d × ...
```
或使用统计数据：
- 通用飞机：T/W = 0.15 ~ 0.25
- 支线客机：T/W = 0.25 ~ 0.35
- 战斗机：T/W = 0.6 ~ 1.2
- 无人机（中空长航时）：T/W = 0.1 ~ 0.2

### 3.2 基于约束的确定方法

推重比需满足所有飞行阶段的约束（取最大值）：
- 起飞场长约束
- 爬升梯度约束
- 最大速度约束
- 巡航升限约束
- 机动过载约束

## 4. 进排气系统

### 4.1 进气系统
- 亚声速：皮托管式进气口，总压恢复 > 0.95
- 超声速：多波系进气口（斜激波 + 正激波）
- 埋入式进气口（隐身飞机）

### 4.2 排气系统
- 亚声速：收敛喷管
- 超声速：收敛-扩张喷管
- 红外抑制：S型喷管、引射喷管

## 5. 螺旋桨设计

### 5.1 螺旋桨效率
η_prop 典型值 0.7 ~ 0.85（定距桨）/ 0.8 ~ 0.9（恒速变距桨）

### 5.2 螺旋桨直径初估
```
D = K × (P / (n × ρ))^(1/5)
```
其中 n 为转速（rps）。

## 6. 燃油估算

### 6.1 任务段燃油分配
```
W_fuel = W_TO - W_empty - W_payload
       = W_fuel_mission + W_fuel_reserve
```

### 6.2 各段燃油消耗（Breguet 方程应用）
- 巡航段：R = (V/L/D) × (1/TSFC) × ln(W_i/W_f)
- 待机段：E = (L/D) × (1/TSFC) × ln(W_i/W_f)
