# 气动布局设计

## 1. 翼型选择

### 1.1 翼型几何参数
- 相对厚度 t/c：影响升力和阻力特性
- 弯度（Camber）：影响零升攻角和最大升力系数
- 前缘半径：影响失速特性
- 最大厚度位置：影响临界马赫数

### 1.2 常用翼型系列
| 翼型系列 | 特点 | 适用场景 |
|----------|------|----------|
| NACA 4-digit | 经典、数据丰富 | 低速通用 |
| NACA 5-digit | 高升力设计 | 高升力需求 |
| NACA 6-series | 层流翼型 | 高速巡航 |
| Eppler系列 | 低雷诺数优化 | 小型无人机 |
| SD系列 | 宽升力范围 | 通用无人机 |
| 超临界翼型 | 延迟激波 | 跨声速 |

### 1.3 翼型选择原则
1. 根据设计巡航速度选择翼型系列
2. 高升力需求选用大弯度翼型
3. 高速需求选用薄翼型（t/c < 12%）
4. 注意雷诺数效应的匹配（尤其是小型UAV）
5. 考虑制造工艺对翼型精度的限制

## 2. 机翼设计

### 2.1 机翼平面参数
- **展弦比 AR**：AR = b²/S
  - 高展弦比 → 升阻比高、诱导阻力小（长航时无人机首选）
  - 低展弦比 → 机动性好、结构轻
- **根梢比 λ**：λ = c_tip / c_root
  - 接近 1 → 接近矩形翼，制造简单
  - 约 0.4 → 接近椭圆升力分布（诱导阻力最小）
- **后掠角 Λ**：
  - 无后掠 → 低速优化
  - 后掠角大 → 高亚声速/跨声速延迟激波
- **上反角 Γ**：
  - 正上反角 → 增加横向静稳定性
  - 负上反角（下反角）→ 提高机动性

### 2.2 增升装置
- 前缘缝翼（Slat）：延迟失速
- 后缘襟翼（Flap）：
  - 简单襟翼：ΔC_L,max ≈ 0.9
  - 单缝襟翼：ΔC_L,max ≈ 1.3
  - 双缝襟翼：ΔC_L,max ≈ 1.6
  - 富勒襟翼：ΔC_L,max ≈ 1.8（同时增大翼面积）

### 2.3 机翼面积估算（初步）
基于失速速度约束：
```
S = (2 * W_TO * g) / (ρ * V_stall² * C_L,max)
```
其中 W_TO 为起飞重量，ρ 为空气密度。

## 3. 尾翼设计

### 3.1 水平尾翼
- **尾容量**：V_H = (S_HT * l_HT) / (S_W * MAC)
  - 典型值：0.5 ~ 1.0（常规布局）
- **尾翼面积**：S_HT ≈ 0.15~0.30 × S_W

### 3.2 垂直尾翼
- **尾容量**：V_V = (S_VT * l_VT) / (S_W * b)
  - 典型值：0.03 ~ 0.08
- **尾翼面积**：S_VT ≈ 0.08~0.15 × S_W

## 4. 配平分析

### 4.1 纵向配平
```
C_m_cg = C_m_ac_w + C_L_w * (x_cg - x_ac_w) / MAC
        - η_HT * (S_HT / S_W) * (l_HT / MAC) * C_L_HT = 0
```

### 4.2 静稳定裕度
```
SM = (x_np - x_cg) / MAC
```
- 常规布局典型值：5%~15% MAC
- 放宽静稳定（RSS）：0% ~ -10% MAC（需增稳飞控）

## 5. 阻力估算

### 5.1 零升阻力（寄生阻力）
```
C_D0 = Σ (C_f × FF × Q × S_wet_i) / S_ref
```
- C_f：摩擦系数（由 Re 数和表面粗糙度决定）
- FF：形状因子
- Q：干涉因子（通常 1.0~1.2）

### 5.2 诱导阻力
```
C_Di = C_L² / (π × AR × e)
```
- e：奥斯瓦尔德效率因子（通常 0.7~0.85）

### 5.3 压缩性阻力
跨声速区阻力发散马赫数（M_DD）估算：
- Korn公式：M_DD + (t/c)/cosΛ + C_L/(10×cos²Λ) = 常数
- 超临界翼型可将 M_DD 提高 0.05~0.10

## 6. 升阻特性

典型巡航状态：
- 低速通用飞机：L/D_max ≈ 10~15
- 滑翔机：L/D_max ≈ 30~60
- 长航时无人机：L/D_max ≈ 20~30
- 民航客机：L/D_max ≈ 15~20
- 战斗机：L/D_max ≈ 8~12
