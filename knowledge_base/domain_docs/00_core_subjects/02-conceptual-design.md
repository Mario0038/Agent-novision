# 概念设计方法

## 1. 概念设计的核心任务

概念设计阶段需要确定飞行器的：
1. **总体布局**（构型选择）
2. **主要参数**（起飞重量、翼面积、推力）
3. **外形轮廓**（三面图级别）
4. **可行性判断**（是否满足TLARs）

## 2. 构型选择

### 2.1 常规布局
**特征**：机翼在前，水平尾翼在后，垂直尾翼在后
- 优点：技术成熟，稳定性好，设计风险低
- 缺点：配平阻力较大
- 代表机型：波音737、C-130、MQ-9 Reaper

### 2.2 鸭式布局（Canard）
**特征**：前翼（鸭翼）在前，主翼在后
- 优点：配平升力为正，可减小翼面积
- 缺点：失速特性复杂，前翼尾流影响主翼
- 代表机型：歼-10、阵风、Long-EZ

### 2.3 飞翼布局（Flying Wing）
**特征**：无机身机翼之分，整机即翼
- 优点：隐身性能极佳，升阻比高
- 缺点：操纵性差，稳定性挑战大
- 代表机型：B-2、RQ-170、X-47B

### 2.4 双尾撑布局（Twin Boom）
**特征**：双尾撑连接尾翼组
- 优点：后置推进方便，尾部空间充裕
- 缺点：结构重量较大
- 代表机型：MQ-9 Reaper、P-38

### 2.5 垂直起降（VTOL）布局
**特征**：可实现垂直/短距起降
- 子类：倾转旋翼（V-22）、倾转机翼、升力风扇（F-35B）、多旋翼
- 优点：不依赖跑道
- 缺点：悬停效率低，过渡段控制复杂

### 2.6 混合布局（Blended Wing Body, BWB）
**特征**：机身与机翼融合过渡
- 优点：升阻比极高，内部空间大
- 缺点：增压舱设计困难，旅客逃生问题
- 代表机型：X-48B（验证机）

## 4. 概念设计工具

### 4.1 设计计算表（Spreadsheet）
使用 Excel/Python 建立参数化设计模型，快速迭代

### 4.2 初步外形生成
- OpenVSP（开源飞行器外形参数化工具）
- RDS（Raymer Design System）

### 4.3 快速分析方法
- AVL（Athena Vortex Lattice）— 涡格法气动分析
- XFLR5 — 低雷诺数翼型分析
- 经验公式与统计回归

## 5. 方案筛选方法

### 5.1 Pugh 矩阵
以基准方案为参照，对各方案在各项指标上进行 +/-/0 评价

### 5.2 TOPSIS 法
多属性决策方法，将定性评估转化为定量排序

### 5.3 风险矩阵
评估各方案的技术风险（概率 × 影响程度）
